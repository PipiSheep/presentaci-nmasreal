Hablemos sobre nuestra aplicacion de gestión




Principales funcionalidades


Gestionar todos las entidades que tenemos como clientes, así como los presupuestos que nos solicitan, así como las facturas emitidas.

Principal característica


Estará programada en JAVA un idioma multiplataforma, gracias a su JVM.
Java es un idioma perfecto para una pequeña aplicación que gestione los recursos necesarios




Gestión de información


En dicha aplicación habrá una pequeña bbdd donde guardaremos el historial de clientes, donde podremos hacer gestiones, como ver que entidad es la que mas productos nos ha encargado, que producto es el que mas se vende...

Emisión de Presupuestos


Una vez dado de alta un cliente, y este nos solicita un presupuesto de los productos deseados, el programa emite un presupuesto con los productos seleccionados, listo para imprimir y presentarlo al cliente.

Emisión de facturas


Una vez el cliente haya aceptado el presupuesto, se da paso a la facturación, factura que es producida por el programa automaticamente se valide un presupuesto.
